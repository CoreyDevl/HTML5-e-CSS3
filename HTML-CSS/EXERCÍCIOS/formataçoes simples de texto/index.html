Texto Marcado!

podemos criar também um texto marcado usando a tag <mark> e o texto vai aparecer marcado .
Acontece que, a tag <mark> pode ser acrescida de detalhe da cor do marcador e pra isso vamos acrescentar algumas informações na tag, que sugere a complementação automaticamente, resultando na tag: <mark style="cor" > - sendo esta última uma marcação já com interação em CSS.

Texto grande / Texto PEQUENO

Existe uma questão acerca da tag <big>, que deixa de ser importante uma vez que seu efeito é meramente visual, feito portanto no CSS3. Contudo, é no mínimo curioso o fato de que sua tag oposta: a tag <small> seja estimulada e atualizada. Minha conclusão pessoal é que a solução para ambos os casos seja mesmo o uso das CSS.
