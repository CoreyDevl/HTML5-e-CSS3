Inserindo imagens Dinâmicas

Para usar imagens de maneira a poder parametrizar sua apresentação na tela usaremos uma tag inicial <picture>. Imagens podem ter tamanhos variados e mesmo assim aplicarmos a elas parâmetros de otimização.
Então dentro da tag pitcure vamos adicionar uma tag de parâmetro que é: <source>, com seu respectivos atributos:

  • media - Isso irá indicar o valor a ser exibido no navegador
  • scrset - Esste será o nome do arquivo da imagem
  • type - Essa será a indicação do media type

Deeeepois, e ainda dentro da tag <picture> é que vamos gerar a tag de carregamento da imagem, como já conhecíamos <img>, com seus conhecidos parâmetros: 'src' e 'alt'.

A tag <audio>, vai funcionar como a anterio <picture>, ou seja, permite aplicação de atributos:

    PRELOAD:
    • metadata - carrega apenas informação do audio
    • none - nao carrega nada até que se clik em play
    • auto - já carrega e toca o som direto
  • CONTROLS - se for inserido dentro da tag irá habilitar o player na tela
  • AUTOPLAY - se for inserido na tag irá carregar e reproduzir audio assim que a página for carregada
  • LOOP - se for inserido na tag irá loopar sempre o audio na pagina
