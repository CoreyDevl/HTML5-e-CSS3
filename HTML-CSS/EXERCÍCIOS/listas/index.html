Listas Ordenadas (OL)

Listas ordenadas são geradas à partir da tag <ol>. Listas ordendas possuem sucessão progressiva de valores, e estes podem ter dois tipos de parâmetros:
type: 1 - imposta marcadores numéricos
A - Utiliza marcadores com letras maiúsculas
a - utiliza marcadores com letras minúsculas
I - Utiliza marcadores com números romanos maiúsculos
i - utiliza marcadores com números romanos minúsculos
Logo em seguida o parâmetro opcional "start" seguido do valor por onde a contagem deve começar.
exemplos:

  1. O começo
  2. o meio
  3. o fim

Listas não Ordenadas (UL)

Listas não ordenadas podem incluir itens os quais não necessitam estar em ordem fixa. Podendo portanto serem marcados apenas por circulos pretos ou vazios ou quadrados pretos. As lista não ordendas são geradas com a tag <ul>.
exemplo:

  • Goiaba
  • morango
  • batata
